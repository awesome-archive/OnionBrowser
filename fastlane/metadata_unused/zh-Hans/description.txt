作为一款真正的使用 Tor 网络的 iOS 免费开源浏览器，Onion 浏览器可帮助您绕过审查，以更加安全隐私的方式访问互联网。

“Onion 浏览器：iOS 上最为物美价廉的 Tor 相关产品” - 《The Daily Dot》，2015年10月14日。

媒体报导：纽约时报、卫报、Salon、TechCrunch、Gizmodo、Boing Boing、Lifehacker、Macworld、Daily Dot 等等。

功能：
• 使用 Tor 浏览互联网。
- 保护你的隐私，免受站点追踪的侵扰
- 在不安全的无线网络和 ISP 下保护你的流量安全
- 支持 HTTPS Everywhere，自动启用 HTTPS 流量
- 访问只能使用 Tor 的 .onion 站点

•打击网络追踪：Onion 浏览器将依据您的偏好屏蔽脚本并自动清除 cookie 和标签页。

• 还有更多功能有效保护您的隐私！

免责声明：
- 访问要比正常网络浏览器和 VPN 要慢，因为你的流量会被 Tor 网络加密和中继。

- 多媒体内容通常会绕过 Tor 并影响你的隐私，因此视频文件和视频直播在 Onion Browser 中会被默认阻拦，不受支持。

- Onion Browser 在受限的网络连接中可能无法使用。如果无法连接，请设置为使用网桥，并退出重进应用。

- 使用 Onion Browser 时责任自负。除了移动设备之外的还有别的隐私数据。
